незарегистрированный пользователь* - это бинарное отношение, связывающее ostis-систему и sc-элемент, обозначающий персону, не прошедшую процедуру регистрации в системе.

Незарегистрированный пользователь имеет доступ на чтение предметной части базы знаний ostis-системы. Может работать с ostis-системы в режиме эксплуатации, т.е. может только задавать запросы, адресуемые предметной части базы знаний (т.е. предметные задачи).

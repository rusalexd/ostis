Задачей агента подсчёта количества логических формул является вычисление количества определений и утверждений. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления количества логических формул. У запроса отсутсвуют аргументы. В ходе выполнения агента явно указывается связь с scp-программой подсчета мощности множества и scp-программой генерации ответа для агентов оценки качества.

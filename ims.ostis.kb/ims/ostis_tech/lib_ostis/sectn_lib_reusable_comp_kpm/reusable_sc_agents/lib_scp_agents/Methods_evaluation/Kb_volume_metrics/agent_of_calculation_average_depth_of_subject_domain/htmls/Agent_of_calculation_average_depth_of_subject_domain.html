Задачей агента вычисления cредней глубины предметных областей является вычисление cредней глубины предметной области. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления cредней глубины предметных областей и предназначен для оценки качества предметной области (метрики глубины). Единственным аргументом запроса является предметная область. В ходе выполнения агента явно указывается связь с scp-программой вычисления всех глубин предметной области, scp-программой подсчета среднего значения и scp-программой генерации ответа для агентов оценки качества.

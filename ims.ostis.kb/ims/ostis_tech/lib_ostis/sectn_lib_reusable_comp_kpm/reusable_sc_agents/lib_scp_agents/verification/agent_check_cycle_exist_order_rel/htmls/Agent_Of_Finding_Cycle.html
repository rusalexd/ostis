Задачей проверки наличия цикла в заданном отношении порядка является поиск цикла для заданного отношения порядка. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки наличия циклов в любом отношении порядка. В данном агенте предусмотренна возможность ограничения области определения до конкретного раздела. Для разработчиков удобней использовать этот агент в рамках какого-то раздела, то есть использовать два входных агрумента (первым аргументом является любое отношение порядка, вторым - раздел, в рамках которого происходит поиск цикла). Для администраторов уместней использовать этот агент в рамках всей базы знаний.

Задачей агента проверки на наличие непустого пересечения у элементов одного разбиения является поиск наличия непустых пересечений у элементов одного разбиения в рамках указанной сущности. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска наличия непустых пересечений у элементов одного разбиения в рамках указанной сущности.

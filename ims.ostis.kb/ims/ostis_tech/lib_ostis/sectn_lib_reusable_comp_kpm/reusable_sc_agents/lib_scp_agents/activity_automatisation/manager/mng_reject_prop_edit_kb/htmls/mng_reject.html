Задачей агента отклонения менеджером предложения по редактированию базы знаний является отклонение* менеджером* исполнителя*, подписавшегося на выполнение проектного задания. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу отклонения менеджером предложения по редактированию базы знаний. Единственным аргументом запроса является знак действия. рассмотреть предлагаемого исполнителя проектной задачи..

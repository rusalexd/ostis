Задачей агента поиска всех нерассмотренных предложений конкретного разработчика является поиск всех нерассмотренних предложений разработчика. На вход данному агенту подается разработчик. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска всех нерассмотренных предложений разработчика. Результатом выполнения агента является вывод на экран всех нерассмотренных предложений разработчика.

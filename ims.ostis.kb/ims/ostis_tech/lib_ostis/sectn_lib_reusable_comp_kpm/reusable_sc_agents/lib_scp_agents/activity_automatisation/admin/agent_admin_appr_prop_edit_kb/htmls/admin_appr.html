Задачей агента утверждения администратором предложения по редактированию базы знаний является утверждение* администратором* предложения*, описывающего изменения, вносимые разработчиком* в базу знаний. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу утверждения администратором предложения по редактированию базы знаний. Единственным аргументом запроса является знак действия. рассмотреть предложение по редактированию базы знаний или знак действия. рассмотреть новое проектное задание.

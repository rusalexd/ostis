Задачей агента поиска всех инициированных знаков действий является поиск всех инициированных знаков действий. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска всех инициированных знаков действий. У данного запроса нет аргументов. Результатом выполнения агента является вывод на экран всех инициированных знаков действий, предложений в рамках этих действий, если такие есть, а также авторов данных предложений.

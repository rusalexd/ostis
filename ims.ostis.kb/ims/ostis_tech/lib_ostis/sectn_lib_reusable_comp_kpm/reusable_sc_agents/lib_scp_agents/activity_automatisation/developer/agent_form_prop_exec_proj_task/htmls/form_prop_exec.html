Задачей агента формирования предложения исполнителя проектного задания является формирование предложения* исполнителя* проектного задания. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу формирования предложения исполнителя проектного задания. Первым аргументом запроса является знак действия, соответствующий проектному заданию, вторым аргументом - знак зарегистрированного пользователя*, предлагаемого в качестве исполнителя*.

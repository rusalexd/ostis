Задачей агента поиска всех запросов конкретного разработчика является поиск всех предложений разработчикa. На вход данному агенту подается разработчик. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска всех предложений разработчика. Результатом выполнения агента является вывод на экран всех предложений разработчикa.

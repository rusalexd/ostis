Конкретные классы действий в рамках Русского языка именуются по следующим правилам:

  • в начале идентификатора пишется слово действие и ставится точка;
  • далее со строчной буквы идет либо содержащее глагол совершенного вида в инфинитиве описание сути того, что требуется получить в результате выполнения действий данного класса, либо вопросительное предложение, являющееся спецификацией запрашиваемой (ответной) информации.

Например:

действие. сформировать полную семантическую окрестность указываемой сущности

действие. верифицировать заданную sc-структуру

Допускается использовать менее строгие идентификаторы, которые, однако, обязаны оперировать словом действие и достаточно четко специфицировать суть действий описываемого класса.

Например:

действие редактирования базы знаний

действие, направленное на установление темпоральных характеристик указываемой сущности

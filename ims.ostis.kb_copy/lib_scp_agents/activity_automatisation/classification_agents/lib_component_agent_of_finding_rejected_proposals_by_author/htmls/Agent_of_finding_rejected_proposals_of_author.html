Задачей агента поиска всех отклоненных предложений разработчика является поиск отклоненных предложений разработчика. На вход данному агенту подается разработчик. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска всех отклоненных предложений разработчика. Результатом выполнения агента является вывод на экран всех отклоненных предложений разработчика.

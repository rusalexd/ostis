Задачей агента поиска отклоненных пользователем запросов является поиск отклоненных пользователем предложений. На вход данному агенту подается пользователь.Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска всех отклоненных конкретным пользователем предложений.Результатом выполнения команды является вывод на экран всех отклоненных конкретным пользователем предложений.

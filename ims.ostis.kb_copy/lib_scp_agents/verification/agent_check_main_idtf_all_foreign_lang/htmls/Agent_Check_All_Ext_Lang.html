Задачей агента проверки наличия основных идентификаторов сущности для всех внешних языков является поиск элементов из предложенного контура, у которых нет основных идентификаторов хотя бы на одном из внешних языков системы. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки основных идентификаторов сущности для всех внешних языков. Единственным аргументом запроса является знак структуры, элементы которой проверяются.

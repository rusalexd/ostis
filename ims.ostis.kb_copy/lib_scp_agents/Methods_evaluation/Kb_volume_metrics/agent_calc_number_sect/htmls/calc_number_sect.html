Задачей агента подсчета количества разделов базы знаний является подсчет количества разделов базы знаний. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу подсчета количества разделов базы знаний. У данного запроса нет аргументом. В ходе выполнения агента явно указывается связь с scp-программой подсчета мощности множества и scp-программой генерации ответа для агентов оценки качества.

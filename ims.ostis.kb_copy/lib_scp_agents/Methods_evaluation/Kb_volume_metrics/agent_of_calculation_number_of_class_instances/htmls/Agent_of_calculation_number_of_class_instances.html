Задачей агента подсчета количества элементов класса является подсчет всех элементов у абсолютного понятия и его потомков. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления количества элементов класса. Единственным аргументом запроса является абсолютное понятие. Результатом работы агента является мощность данного абсолютного понятия.

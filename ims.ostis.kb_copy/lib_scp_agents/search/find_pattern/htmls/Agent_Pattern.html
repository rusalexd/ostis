Задачей агента поиска конструкций для заданного шаблона является нахождение конструкций, удовлетворяющих заданному шаблону. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска конструкций для заданного шаблона. Аргументом запроса является шаблон, для которого должны быть найдены все соответствующие ему конструкции.
